## JSON

## JSON values

A JSON value can be:
- numbers `1`
- boolean `true/false`
- strings `""`
- objects `{}`
- arrays `[]`


## Example

``` json
{
  "model": "iPhone X",
  "price": 1200.00,
  "owner": {
    firstName: "Daniel",
    lastName: "Simionescu"
  },
  "cameras": [
    "front": 7,
    "back": 12
  ]
}
```

::: danger Remember
JSON strings use double quotes `" "` and not single quotes or apostrophies `' '`.
:::
